Дисконт в теории игр

В теории игр, дисконт — текущая стоимость денежной единицы будущего периода, умноженная на вероятность повторения игры для игр с неизвестным числом повторений.

Залоговый дисконт

Разница между реальной рыночной стоимостью залогового имущества и его залоговой стоимостью, используемой для определения суммы выдаваемого банком кредита.

Дисконтирование денежных потоков

Метод оценки инвестиций, при котором будущие денежные потоки приводятся к текущей стоимости с использованием ставки дисконтирования, отражающей риск и стоимость капитала.

Ставка дисконтирования

Процентная ставка, используемая для дисконтирования будущих денежных потоков к их текущей стоимости. Отражает требуемую доходность инвестора с учетом риска инвестиции.

Дисконтный брокер

Брокерская компания, предлагающая услуги по торговле ценными бумагами с существенно сниженными комиссионными сборами по сравнению с традиционными брокерами. Часто предоставляет ограниченный набор дополнительных услуг.

Премия за риск

Дополнительная доходность, которую инвесторы ожидают получить в качестве компенсации за принятие на себя более высокого уровня риска по сравнению с безрисковыми инвестициями, такими как государственные облигации.
